# 🧙‍♀️ FantasyVult

**FantasyVult** — это компьютерное приложение-каталог по 14 фэнтези-вселенным с тестами, избранным и интересными фактами.  
Проект разработан как **курсовая работа** и одновременно как первый опыт в работе с **Qt Creator**, **SQLite**, **Git** и **структурой реального приложения**.

---

### ✨ Особенности приложения

- 📚 **Каталог фэнтези-вселенных** с карточками, изображениями и описаниями
- 🧪 **Интерактивные тесты** по каждой вселенной (по 10 вопросов)
- ❤️ Возможность **добавлять вселенные в “Любимое”**
- 📜 **Интересные факты** с изображениями по каждой вселенной
- 🔍 **Поиск и фильтрация** по вселенным
- 👤 **Регистрация и авторизация**
- 📖 Встроенное **руководство пользователя**

### 🛠 Технологии

- 💻 **Qt 6.9 (C++ / Widgets)**
- 🗂 **SQLite** для хранения данных
- 🖼 **QPixmap + обычная файловая система** (также используется `.qrc`)
- 🧪 **QDebug** для отладки

---

### ⚙️ Как запустить проект

1. **Откройте файл проекта** `FantasyVult.pro` в Qt Creator.
2. Убедитесь, что структура папок соблюдена:
3. **Соберите и запустите проект** (`Ctrl + R` в Qt Creator).
