# FantasyVult  

**FantasyVult** — это настольное приложение‑каталог, посвящённое четырнадцати популярным фэнтези‑вселенным. Программа разработана в рамках выполнения курсовой работы по дисциплине **«Основы алгоритмизации и программирования»** и демонстрирует навыки объектно‑ориентированного программирования на C++, работу с фреймворком **Qt**, а также проектирование структуры данных и пользовательского интерфейса.

Приложение включает каталог вселенных, систему тестирования, механизм избранного, модуль интересных фактов и встроенное руководство пользователя. Проект отражает полный цикл разработки: от проектирования структуры данных до реализации интерфейса и логики взаимодействия.

---

## Цели и задачи проекта
В ходе разработки были поставлены и реализованы следующие задачи:

- создание структуры данных для хранения информации о фэнтези‑вселенных, тестах и фактах;
- разработка графического интерфейса на основе Qt Widgets;
- реализация механизма регистрации и авторизации пользователей;
- создание каталога с возможностью поиска и фильтрации;
- разработка системы тестирования (по 10 вопросов для каждой вселенной);
- реализация функционала «Избранное»;
- интеграция изображений и медиа‑контента через файловую систему и ресурсы `.qrc`;
- обеспечение стабильной работы приложения и отладка с использованием `QDebug`.

---

## Функциональные возможности

### 1. Каталог фэнтези‑вселенных
- карточки с изображениями, описаниями и ключевой информацией;
- удобная навигация по списку;
- фильтрация и поиск по названию.

### 2. Интерактивные тесты
- тесты по каждой вселенной (10 вопросов);
- автоматическая проверка результатов;
- отображение итогового балла.

### 3. Система «Избранное»
- добавление и удаление вселенных из списка избранных;
- сохранение состояния между сессиями.

### 4. Интересные факты
- подборка фактов по каждой вселенной;
- изображения и краткие пояснения.

### 5. Пользовательские функции
- регистрация и авторизация;
- хранение пользовательских данных в базе SQLite;
- встроенное руководство пользователя.

## Используемые технологии

### Язык программирования
- **C++ (ООП, STL, Qt API)**

### Фреймворк
- **Qt 6.9 (Widgets)**

### Работа с данными
- **SQLite**  
- файловая система + ресурсы `.qrc`  
- загрузка изображений через `QPixmap`

### Инструменты разработки
- Qt Creator  
- Git и GitHub  
- QDebug для отладки  

---

## Установка и запуск

1. Откройте файл проекта  
в **Qt Creator**.

2. Убедитесь, что структура каталогов и ресурсов сохранена.

3. Соберите и запустите проект:  
**Ctrl + R** в Qt Creator.
